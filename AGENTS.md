# AGENTS.md

## How to run tests
- Python: `pip install -e .[dev]` then `pytest`
- JS: `npm install` then `npm test`

## Project layout
- src/      – main library code
- tests/    – unit tests
- docs/     – design notes

## Rules
- Always run tests before proposing a PR.
- Prefer small, focused changes over big refactors.
